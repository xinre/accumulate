1. Hadoop 不是微服务必备的

┌─────────────────────────────────────────────────────────────────┐
│                      Hadoop 用在哪？                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Hadoop生态:                                                   │
│   ┌─────────────────────────────────────────────────────┐      │

│   └─────────────────────────────────────────────────────┘      │
│                                                                 │
│   微服务改造需要的是:                                            │
│   ┌─────────────────────────────────────────────────────┐      │

│                                                                 │
│   除非你们有大数据需求，否则 Hadoop 不用加                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘


2. Zookeeper 现在很少单独用了

┌─────────────────────────────────────────────────────────────────┐
│                    Zookeeper 去哪了？                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   以前:                                                         │
